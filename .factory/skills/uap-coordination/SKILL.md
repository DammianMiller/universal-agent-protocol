# UAP Agent Coordination Skill

## Commands

### Agent Status
```bash
uap agent status
```
Shows all registered agents and their current status.

### Dashboard
```bash
uap dashboard
```
Shows the UAP session dashboard with live telemetry.

### Dashboard Views
```bash
uap dashboard --view summary
uap dashboard --view snapshot
```
