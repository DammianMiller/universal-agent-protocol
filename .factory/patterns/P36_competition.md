# P36: Competition Domain Research

**Category**: Research
**Abbreviation**: Competition-Research

## Pattern

For competition/challenge tasks, research domain-specific best practices before implementing.

## Rule

```
Competition task → Research winning strategies → Implement optimal approach.
```

## Implementation

1. Identify competition domain
2. Research state-of-the-art approaches
3. Implement best-known strategy
4. Iterate to improve

## Domain Research Sources

- Competition leaderboards
- Academic papers
- Open-source implementations
- Domain forums/communities

## Example Domains

- Code golf: Research language tricks
- Algorithm competitions: Known algorithms
- Game AI: Research winning strategies
- Data science: Kaggle kernels

## Anti-Pattern

❌ Starting from scratch without research
❌ Ignoring established best practices
❌ "My approach should work" without validation
