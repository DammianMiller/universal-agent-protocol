# UAP Benchmark - REAL Harbor Execution Only

## ✅ Current State: Real Terminal-Bench via Harbor

All legacy simulation scripts removed. Only real Harbor-based benchmarking remains.

## Files for Benchmarking

| File | Purpose |
|------|---------|
| `run_harbor_benchmark.sh` | Main runner script |
| `harbor-configs/quick-suite.yaml` | 4-task benchmark config |
| `tools/agents/uam_agent.py` | Harbor agent implementation |
| `BENCHMARK.md` | Usage documentation |

## Usage

```bash
# Quick test (1 task):
./run_harbor_benchmark.sh --quick-test

# Full suite (4 tasks - REAL Terminal-Bench execution):
./run_harbor_benchmark.sh --full
```

## How It Works

1. Harbor creates isolated Docker container
2. UAM agent installed in container via `BaseInstalledAgent` interface
3. Task instruction passed to Qwen3.5 API endpoint  
4. Verification script runs inside container
5. Results reported based on actual completion

## Prerequisites

- Harbor CLI: `pip install harbor-framework`
- Docker daemon running
- Qwen3.5 API accessible at configured endpoint
